# Library Management System (CI Demo)

This project is used to demonstrate a Jenkins CI pipeline.

## Features
- Simple Python file execution
- Jenkins Freestyle and Pipeline jobs
- GitHub integration
- Auto build using Poll SCM

## Files
- new.py – Python script
- README.md – Project documentation

## How Jenkins Works
1. Jenkins pulls the code from GitHub.
2. Jenkins runs the Python file.
3. Jenkins shows output in the console.
4. Jenkins auto-builds when the code changes.
